# 双黑洞正面碰撞模拟与引力波提取

## 作业内容

1. **引力波模式图**  
   - `GW_mode_20.png`：\(l=2,m=0\) 模式的引力波信号  
   - `GW_mode_22.png`：\(l=2,m=2\) 模式的引力波信号  

2. **演化过程视频**  
   - `BinaryBH_Short.mp4`：利用 hdf5 文件制作的演化视频  

3. **原始数据文件夹**  
   - `data/`：包含 GRChombo 运行产生的所有输出文件（检查点、绘图数据、引力波提取数据等）

---

## 模拟配置

- 等质量双黑洞正面碰撞（质量各 0.5，初始间距 20）  
- 采用 GRChombo 进行数值模拟  
- 提取引力波模式 \(l=2,m=0\) 和 \(l=2,m=2\)  
- 使用 Visit 制作演化视频
